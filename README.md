# Telegram Payment & Subscription Bot

This is a Telegram bot that manages subscriptions, payments, and user requests.

## Features
- `/star` - Greets the user
- `/join` - Request to join a private group
- `/cancel` - Cancel an ongoing action
- `/leave` - Request to leave the group
- `/broadcast` (Admin) - Send a message to all users
- `/ping` (Admin) - Check bot status
- `/send <user_id>` (Admin) - Send a message to a user
- `/about` - Shows bot creator info
- `/help` - Lists all available commands
